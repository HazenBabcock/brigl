BRIGL
=====

BRIGL is a javascript program that reads and parses `LDraw <http://www.ldraw.org/>`_ models, and
renders them with WebGL using the `three.js <https://threejs.org/>`_ library.

The source code is available on `Github <https://github.com/HazenBabcock/brigl>`_.
